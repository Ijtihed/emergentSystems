Origin

Google Research, Differentiable Logic Cellular Automata, Mordvintsev et al., March 2025. This demo uses a genetic algorithm to approximate the gradient-based rule learning approach from the paper.

How It Works

Binary CA with outer-totalistic rules. 18 bits define how each cell updates based on its state and neighbor count. A GA evolves populations of rules, testing which produce patterns closest to your target.

Applications

Automated discovery of self-organizing rules for pattern generation and engineering. Bridges neural networks and symbolic computation, specify what you want and let the system find how.
