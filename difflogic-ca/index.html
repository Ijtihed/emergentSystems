DiffLogic Cellular Automata info

Origin

Inspired by Google Research's "Differentiable Logic Cellular Automata" (March 2025), where discrete binary logic gate rules are learned via gradient descent to produce target patterns. The original paper uses differentiable relaxation; this demo uses a genetic algorithm to search the rule space — capturing the same idea of reverse-engineering CA rules from desired patterns.

Google Self-Organising Systems team (Alexander Mordvintsev et al.), 2025. No public interactive playground exists for the original paper yet.

How It Works

A cellular automaton is a grid where each cell's next state depends only on its current state and its neighbors. Here, rules are "outer-totalistic" — the next state depends on the current state (alive/dead) and the count of alive neighbors (0-8). That's just 18 binary decisions defining a complete rule. A population of random rules is evolved: each is tested by running it from random initial conditions and comparing the result to your target pattern. The best rules survive, crossover, and mutate over generations.

Applications

Automated discovery of self-organizing rules for engineering, pattern generation, and texture synthesis. Understanding how simple local rules produce global order. The broader DiffLogic approach enables learning discrete computation from continuous optimization — bridging neural networks and symbolic AI.

How To Use

Draw a target pattern on the left canvas (click and drag). Hit learn to start the genetic algorithm — watch the right canvas show the best rule found so far. Click run to animate the learned rule continuously from random initial conditions. Use random to generate an interesting target pattern automatically.
