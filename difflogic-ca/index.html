Origin

Inspired by Miotti, Niklasson, Randazzo, Mordvintsev at Google, March 2025. arXiv. The paper trains logic gate circuits via gradient descent on continuous relaxations to learn CA rules. This demo uses evolutionary search as a browser-accessible alternative.

How It Works

The paper uses multi-bit cell states and networks of 16 possible logic gates trained end-to-end. This demo simplifies to 1-bit cells with outer-totalistic rules, where a genetic algorithm evolves the 18-bit rule table by testing which rules keep your target pattern stable under iteration.

Applications

The full DiffLogic approach enables learnable programmable matter and robust self-healing circuits. This demo explores the core idea, reverse-engineering local rules from desired global patterns, which is the central question the paper addresses.
