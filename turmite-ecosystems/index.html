← back Turmite Ecosystems

Origin

Extension of Langton's Ant and Ed Pegg's turmite survey. Multiple ants with different rule tables sharing one grid. The multi-ant interaction space is essentially unmapped.

How It Works

Define 2 to 3 turmite species, each with its own state transition table. Drop colonies of each on the grid. One species builds highways, another builds spirals. Where territories collide you get interference patterns.

Applications

Emergent territory formation, higher-order automata from simple agents, visual exploration of unmapped multi-agent parameter space.

Select a species and click to place a colony. Watch territories collide and produce interference patterns.
Presets per species A, B, C include langton, highway, spiral, chaotic. Click mode place A, place B, place C. Speed slider, reset.
